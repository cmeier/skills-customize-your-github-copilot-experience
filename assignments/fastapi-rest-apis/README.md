# 📘 Assignment: FastAPI REST APIs

## 🎯 Objective

Learn to build scalable RESTful web services using the FastAPI framework. Students will develop practical skills in creating API endpoints, handling HTTP requests and responses, and managing data validation and error handling.

## 📝 Tasks

### 🛠️ Create Basic API Endpoints

#### Description
Build a FastAPI application with basic GET and POST endpoints to manage a simple data resource.

#### Requirements
Completed program should:

- Set up a FastAPI application with a main entry point.
- Create a GET endpoint that returns a list of items or a welcome message.
- Create a POST endpoint that accepts JSON data and stores it.
- Include proper HTTP status codes for successful and failed requests.
- Test endpoints using curl, Postman, or FastAPI's built-in Swagger UI.

### 🛠️ Add Data Validation and Error Handling

#### Description
Enhance the API with request validation using Pydantic models and implement comprehensive error handling.

#### Requirements
Completed program should:

- Define Pydantic models for request and response data.
- Validate incoming JSON data and return appropriate error messages for invalid inputs.
- Implement custom error responses with meaningful error messages.
- Add a PUT or DELETE endpoint for updating or removing items.
- Include proper HTTP status codes (200, 201, 400, 404) for different scenarios.
